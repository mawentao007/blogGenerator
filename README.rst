静态博客辅助工具
================


1.index页面内容部分生成

2.边栏索引部分生成

3.自动扫描源文件目录postsrc，生成文件列表postname.xml

4.简单版本控制，根据修改时间决定是否重新生成目录正文

使用方法
================

配置目录 ::

         config目录下面有pathfile.xml文件，负责指定生成的目标路径，源文件路径和模板路径;


2.写博客 ::

  在postsrc(默认)目录下写博客，按照模板格式;

3.执行 ::

  在根目录下执行run文件，参数为new的时候表示生成所有博客内容。
